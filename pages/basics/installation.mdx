# __Installation__

<Callout type="warning">
Node.js **16.11.0** or newer is required.
</Callout>

To install **riffy**, run the following command in your terminal:
```bash copy
npm install riffy
```

### __Optional Package__
- **[musicard](https://www.npmjs.com/package/musicard)** for making your discord music bot more visual.

## __Importing__

To import **Riffy**, use the following code:
```js copy filename="index.js"
const { Riffy } = require("riffy");
```
